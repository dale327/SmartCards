SmartCards - cервис для изучения карточек

Приложение позволяет создавать учебные колоды и эффективно запоминать информацию.

Основные возможности: создание и редактирование тематических колод, добавление карточек с поддержкой загрузки
изображений,
автоматический перевод слов через внешнее API, адаптивный интерфейс на Bootstrap.

Стек технологий:
Python / Flask
SQLAlchemy (ORM)
Bootstrap 5
Flask-WTF
